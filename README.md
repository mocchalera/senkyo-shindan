# 衆議院選挙2026 投票先診断ウェブアプリ

2026年2月8日（日）投開票の第51回衆議院議員総選挙に向けた、政策一致度ベースの投票先診断アプリです。  
10問の回答から、9政党との一致率を算出し、結果と学びコンテンツを表示します。

## 技術構成

- 静的 `HTML + CSS + JavaScript`（バニラ）
- 単一ファイル構成（`index.html`）
- 外部依存なし
- モバイルファースト / レスポンシブ対応
- ダークモードUI

## ファイル構成

```text
.
├── index.html
└── README.md
```

## 実装内容

- ウェルカム画面
  - タイトル、選挙日表示、注意書き、診断スタート
  - ゲーム感のあるウォームアップ演出
- 質問画面（10問）
  - 争点別の選択式質問
  - プログレスバー（`10問中 X問目`）
  - 回答時リアクション表示
- スコアリング
  - 各選択肢に対し9政党へ0〜3点を付与
  - 一致率は「得点 / 政党ごとの理論最大点」で算出
- 結果画面
  - TOP3政党表示
  - 全9政党の一致率バーチャート
  - 政党別サマリーと一致争点ハイライト
  - SNSシェア、再診断
- 学びモード
  - 争点ごとの解説（なぜ重要か）
  - ユーザー回答と各政党スタンスの比較
  - 政党ポジションマップ（経済軸×社会軸）
  - 各政党の公約ハイライト

## 使い方

1. `index.html` をブラウザで開く
2. 「診断スタート」を押して10問に回答
3. 結果サマリーと学びモードを確認

## 注意事項

- 本アプリは政策の一致度を示す参考ツールです。
- 実際の投票判断では、候補者情報・地域事情・最新の公約情報もあわせて確認してください。
- 特定の政党への投票を推奨・強制するものではありません。
